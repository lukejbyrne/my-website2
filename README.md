# my-website2

Personal website for [Luke Byrne](https://youtube.com/@AI_LUKE) — creator, builder, and founder of Byrne Media.

Static site built with plain HTML, CSS, and JS. No framework, no build step.

## Pages

- **Landing page** — hero, about, work, and contact sections
- **Blog** — listing page + 6 individual posts on AI tools, Claude Code, MCP, and the creator journey

## Local development

```
npx serve -p 4000 .
```

## Design

- Dark mode by default, light mode toggle (persisted via localStorage)
- Instrument Serif + DM Sans from Google Fonts
- Coral accent colour (`#e8654a`)
